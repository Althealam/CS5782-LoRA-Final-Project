# LoRA Project Starter

This starter folder is a clean starting point for a course re-implementation of LoRA on `roberta-base` and GLUE.

## Files
- `PROPOSAL_DRAFT.md`: a proposal draft you can edit and submit
- `EXECUTION_PLAN.md`: a practical plan from setup to experiments
- `START_HERE_COLAB_GITHUB.md`: step-by-step setup guidance
- `requirements.txt`: starter dependencies
- `src/lora.py`: starter LoRA module
- `scripts/train_glue.py`: starter training script

## Recommended First Experiment
- Dataset: `sst2`
- Methods: frozen baseline, full fine-tuning, LoRA
- Model: `roberta-base`

## Notes
- The LoRA implementation here is intentionally simple and readable.
- This is a starter, not a finished reproduction.
